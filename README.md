# Delivery Dash (브라우저 미니게임)

**컨셉:** 택배를 주워서 목적지에 배달하는데, 추격자(드론)를 피하면서 점수를 올리는 2D 탑다운 액션.

## 조작
### 모바일(추천)
- 이동: **왼쪽 화면 드래그(가상 조이스틱)**
- 대시: **오른쪽 화면 탭**
- 시작/재시작: **탭**

### 키보드
- 이동: WASD / 방향키
- 대시: Space (쿨다운)
- 일시정지: P
- 시작/재시작: Enter

## 실행
가장 간단:

```bash
cd games/delivery-dash
python3 -m http.server 5173
# 브라우저에서 http://localhost:5173
```

(로컬 파일로 열어도 대체로 되는데, 브라우저 보안 정책에 따라 제한될 수 있어서 http 서버 추천)

## 현재 구현
- **60초 미니런**: 패키지 줍기 → 배달 지점에 들어가면 배달 성공
- 배달할수록 난이도 증가(드론 가속/추가)
- **배달 후 업그레이드 3개 중 1개 선택** (키보드: 1~3 / 모바일: 카드 탭)
- 적 타입:
  - Chaser: 기본 추격
  - Sniper: 거리 유지 + 탄환 발사
  - Charger: 일정 주기 돌진
- 지뢰(Mine) 등장
